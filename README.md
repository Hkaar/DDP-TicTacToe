A TicTacToe game programmed in C!

## Features

- 3 Tiered difficulty AI
- Fully functioning implementation of Tic Tac Toe
- Cross platform (Linux & Windows)

## Requirements

- GCC / MSVC / Clang
- CMake >= 3.12

## Build Guide

Clone the project

```bash
git clone "https://github.com/Hkaar/DDP-TicTacToe.git"
```

Go to the project directory

```bash
cd DDP-TicTacToe
```

Run the build script

`command prompt`

```batch
./build.bat
```

`bash`

```bash
./build.sh
```

Run the executable

`command prompt`

```batch
./run.bat
```

`bash`

```bash
./run.sh
```
